## Solution:

Since the file given to us is an apk file, I looked up how apk files are usually analyzed and installed jadx, which can be used to decompile apk files.

Analyzing MainActivity, we see

```
package byuctf.downwiththefrench;

import android.os.Bundle;

/* loaded from: classes3.dex */
public class MainActivity extends AppCompatActivity {
    @Override // androidx.fragment.app.FragmentActivity, androidx.activity.ComponentActivity, androidx.core.app.ComponentActivity, android.app.Activity
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);
        Utilities util = new Utilities(this);

cleanup() looks a little suspicious so I decided to look into it.

Looking at the cleanup() function, we can conclude that there are 28 parts of the flag. And the function cleanup() wipes the TextViews.
I looked into `res/layout/activity_main.xml` which controls the layout that loads when the app is started.

```
